# Генератор карт на основе шума Перлина

Это приложение с графическим интерфейсом для генерации и визуализации различных карт с использованием алгоритма шума Перлина.

## Возможности

- Генерация карт на основе шума Перлина
- Настройка параметров генерации (размер, масштаб, октавы и др.)
- Создание карт высот с настраиваемым уровнем моря
- Генерация температурных карт
- Просмотр карт в 2D и 3D режимах
- Сохранение сгенерированных карт

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/dmitrym1309/PerlinNoiseGenerator.git
cd PerlinNoiseGen
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

Запустите приложение:
```bash
python Noise/test.py
```

Для создания исполняемого файла:
```bash
python Noise/build.py
```

## Требования

- Python 3.6+
- См. requirements.txt для списка зависимостей
