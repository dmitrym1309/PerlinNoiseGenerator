# Генератор карт на основе шума Перлина

Это приложение с графическим интерфейсом для генерации и визуализации различных карт с использованием алгоритма шума Перлина.

## Возможности

- Генерация карт на основе шума Перлина
- Настройка параметров генерации (размер, масштаб, октавы и др.)
- Создание карт высот с настраиваемым уровнем моря
- Генерация температурных карт
- Просмотр карт в 2D и 3D режимах с настраиваемым качеством
- Сохранение сгенерированных карт в высоком разрешении

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/dmitrym1309/PerlinNoiseGenerator.git
cd PerlinNoiseGen
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

Запустите приложение:
```bash
python Noise/main.py
```

Для создания исполняемого файла:
```bash
python Noise/build.py
```

## Параметры генерации

- **Размер**: размер генерируемой карты (в пикселях)
- **Масштаб**: масштаб шума Перлина (влияет на "зернистость" карты)
- **Seed**: зерно генерации (позволяет воспроизводить одинаковые карты)
- **Октавы**: количество слоев шума (влияет на детализацию)
- **Persistence**: влияние каждой следующей октавы (0-1)
- **Уровень моря**: порог для карты высот (определяет уровень воды)

## Типы карт

1. **Шум Перлина**: базовая карта шума
2. **Карта высот**: карта рельефа с настраиваемым уровнем моря
3. **Температурная карта**: распределение температур на основе высоты

## 3D визуализация

Доступны три уровня качества 3D отображения:
- **Низкое**: быстрая отрисовка, подходит для предпросмотра
- **Среднее**: оптимальный баланс качества и производительности
- **Высокое**: максимальное качество, может быть медленнее

## Управление в 3D режиме

- Вращение: левая кнопка мыши
- Масштабирование: правая кнопка мыши или колесико
- Перемещение: средняя кнопка мыши
