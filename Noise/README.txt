Генератор карт на основе шума Перлина

Возможности:
- Генерация карт различных типов (шум, высота, температура)
- 2D и 3D визуализация
- Настройка всех параметров генерации
- Выбор качества 3D отображения
- Сохранение сгенерированных карт

Управление в 3D режиме:
- Вращение: левая кнопка мыши
- Масштабирование: правая кнопка мыши или колесико
- Перемещение: средняя кнопка мыши

Параметры:
- Размер: размер генерируемой карты
- Масштаб: масштаб шума Перлина
- Seed: зерно генерации
- Октавы: количество слоев шума
- Persistence: влияние каждой октавы
- Уровень моря: порог для карты высот
